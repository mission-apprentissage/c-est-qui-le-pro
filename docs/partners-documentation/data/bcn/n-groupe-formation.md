# Table N_GROUPE_FORMATION
{: .no_toc }

- TOC
{:toc}

La table **N_GROUPE_FORMATION** contient les différentes familles de métiers.

## Périodicité de mise à jour

- Fréquence de mise à jour par le producteur inconnue
- Les données sont mises à jour ponctuellement

## Format des données

Les données sont disponibles au format CSV et JSON.

## Entité responsable

DEPP – Bureau des nomenclatures et répertoires

## Source

- [Page de la table sur le site de la BCN](https://bcn.depp.education.fr/bcn/workspace/documentation/n/N_GROUPE_FORMATION)

## Utilisation

Nous utilisons les données pour permettre de connaitre les différentes familles de métiers.

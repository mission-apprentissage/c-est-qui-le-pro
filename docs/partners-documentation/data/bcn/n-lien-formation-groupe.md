# Table N_LIEN_FORMATION_GROUPE
{: .no_toc }

- TOC
{:toc}

La table **N_LIEN_FORMATION_GROUPE** permet de faire le lien entre une formation et sa famille de métier.

## Périodicité de mise à jour

- Fréquence de mise à jour par le producteur inconnue
- Les données sont mises à jour ponctuellement

## Format des données

Les données sont disponibles au format CSV et JSON.

## Entité responsable

DEPP – Bureau des nomenclatures et répertoires

## Source

- [Page de la table sur le site de la BCN](https://bcn.depp.education.fr/bcn/workspace/documentation/n/N_LIEN_FORMATION_GROUPE)

## Utilisation

Nous utilisons les données pour permettre d'expliciter les liens entre secondes communes et bac pro au sein de familles de métiers.
